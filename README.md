# python
RSA暗号っぽいものです。

素数が何番目かを求める関数を使います
  (例) 1 => 2, 2 => 3, 3 => 5, 4 => 7, 5 => 11
これによって素数ではない数字でも暗号化することができるようになっています。

肝心な暗号の方ですが、入力された文字の他に暗号化強度の分のダミーを追加しています。
ダミーは暗号化強度が増えるほど増えます。よって解読に時間がかかるようになります。

ダミーは素因数分解すると素数が偶数個出てきますが、本物の素数は奇数個出てきます。
  (例) 440657 => 23 * 23 * 17 * 7 * 7
この例だと17が奇数個出てきているので答えは7となります。
